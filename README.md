## 	:toolbox: Getting Started


<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/danimr99/banking-web.git
```

Go to the project directory

```bash
  cd banking-web
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


<!-- Deployment -->
### :triangular_flag_on_post: Deployment

To deploy this project run

```bash
  npm run build
```
